

## Planilha de Controle de Gastos Mensais

### Página Principal - Dashboard
- Resumo do mês atual: total gasto, orçamento restante, barra de progresso do orçamento
- Alerta visual quando os gastos se aproximarem ou ultrapassarem o orçamento definido
- Gráfico de pizza mostrando gastos por categoria
- Gráfico de barras comparando gastos diários/semanais

### Tabela de Gastos
- Tabela estilo planilha com colunas: Data, Descrição, Categoria, Valor
- Botão para adicionar novo gasto rapidamente
- Categorias pré-definidas: Alimentação, Transporte, Moradia, Saúde, Lazer, Educação, Outros (com opção de criar novas)
- Filtros por categoria, período (dia/semana/mês) e busca por descrição
- Opção de editar e excluir gastos

### Configuração de Orçamento
- Definir orçamento mensal total
- Opcionalmente definir limites por categoria
- Indicadores visuais (verde/amarelo/vermelho) conforme nível de gasto

### Armazenamento
- Todos os dados salvos no localStorage do navegador (sem necessidade de login)
- Os dados persistem entre sessões no mesmo navegador

### Navegação entre Meses
- Seletor de mês/ano para visualizar histórico
- Comparação básica entre meses no dashboard
